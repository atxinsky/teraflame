字体资源说明
===========

本文件夹用于存放游戏所需的字体资源，包括：

1. 主要游戏字体
   - main.ttf (主要游戏文本字体)
   - main_bold.ttf (主要游戏文本粗体字体)
   
2. 标题字体
   - title.ttf (游戏标题字体)
   
3. 数字字体
   - digits.ttf (显示筹码、金额等数字的字体)
   
4. 其他字体
   - card.ttf (牌面显示专用字体，包含扑克符号)

注意：字体文件应当使用TTF或OTF格式。使用字体时请确保遵守相应的许可协议。默认情况下，如果未找到指定字体，游戏将回退使用系统默认字体。
